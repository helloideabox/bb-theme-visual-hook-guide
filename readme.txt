Find Beaver Builder Theme hooks quick and easily by seeing their actual locations inside your theme.

== Description ==

Once installed this plugin adds a drop down menu to the admin bar to display Hooks available in the [Beaver Builder Theme](https://www.wpbeaverbuilder.com/wordpress-framework-theme/)
This plugin is based on Christopher Cochran's Genesis Visual Hook Guide plugin.

**Beaver Builder Theme is required.**

== Installation ==

1. Upload the entire `bb-theme-visual-hook-guide` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I have my admin bar disabled, can I still view the hooks without it? =

Sure! The views are triggered by a query string. Simply add ?bbtheme_hooks=show to the end the url.
